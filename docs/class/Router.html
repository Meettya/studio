Class: Router

Defined in: src/core/router.coffee

Overview

Base class for Router (is singleton, should not be reimplemented nor reinstantiated, and probably not direct accessed).

Instance Method Summary

  • # (void) createRoute(id, watchPath) Creates a route, if it already exists throws exception

  • # (void) getRoute(id) Returns a route
  • # (void) send(sender, receiver, message, headers) sends a message to the receivers

# (void) send(sender, receiver, message, headers)

sends a message to the receivers

Examples:

How to send a string(or any primitive) message 

router.send('mySenderActor','myReceiverActor', 'HELLO WORLD')

How to send an object message 

router.send('mySenderActor','myReceiverActor', {hello:'hello',
                               obj:{
                                 count:1
                               }
                             })

Parameters:

  • sender ( String ) — the sender identification
  • receiver ( String ) — the receiver route identification
  • message ( Object ) — the message to be delivered
  • headers ( Object ) — headers for the message (optional)
