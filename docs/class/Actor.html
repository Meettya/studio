# (void) constructor(options)

Constructs a new actor

Examples:

How to instantiate an actor (in Coffescript) 

myActor = new Actor({
            id: 'myActor',
            process:(body,sender,receiver)-> console.log(body)
          })

Parameters:

  • options ( Object ) — the actor options

Options Hash: (options):

  • id ( String ) — actor id (should be unique), when you instantiate an actor you automatically create a stream on the router with this id
  • process ( Function ) — the process function which will be executed for every message

# (void) doProcess(message) Bound

PRIVATE METHOD SHOULD NOT BE CALLED Takes a stream message and open it for the actor process function format. And creates a promise with the result of the message

# (void) addTransformation(funktion)

Transform the message stream, so you can merge other streams, filter, buffer or do anything you can do with a baconjs stream

Examples:

How to apply a filter transformation (in Coffescript) 

myActor.addTransformation((stream)->stream.filter((message)->message.sender == 'otherActor'))

Parameters:

  • funktion ( Function ) — a function which receives the current stream for this actor and return the new transformed stream

# (void) send(receiver, message)

Sends message to an actor

Parameters:

  • receiver ( String ) — the receiver id
  • message ( Object ) — the content of the message (it could be any js object)
