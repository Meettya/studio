Constructor Details

# (void) constructor(options)

Constructs a new actor and automatically starts it

Examples:

How to instantiate an actor (in javascript) 

var myActor = new Actor({
            id: 'myActor',
            process:function(body,sender,receiver){ console.log(body);}
          });

How to instantiate an actor (in Coffescript) 

myActor = new Actor({
            id: 'myActor',
            process:(body,sender,receiver)-> console.log(body)
          })

Parameters:

  • options ( Object ) — the actor options

Options Hash: (options):

  • id ( String ) — actor id (should be unique), when you instantiate an actor you automatically create a stream on the router with this id
  • process ( Function ) — the process function which will be executed for every message
  • initialize ( Function ) — function called after actor creation
  • errorHandler ( Function ) — function called after and error happens on this actor (tipically you want to log the error)

Instance Method Details

# (void) _doProcess(message) Bound

PRIVATE METHOD SHOULD NOT BE CALLED Takes a stream message and open it for the actor process function format. And creates a promise with the result of the message

# (void) addTransformation(funktion)

Transform the message stream, so you can merge other streams, filter, buffer or do anything you can do with a baconjs stream

Examples:

How to apply a filter transformation (in javascript) 

myActor.addTransformation(function(stream){return stream.filter(function(message){return message.sender == 'otherActor';});});

How to apply a filter transformation (in Coffescript) 

myActor.addTransformation((stream)->stream.filter((message)->message.sender == 'otherActor'))

Parameters:

  • funktion ( Function ) — a function which receives the current stream for this actor and return the new transformed stream

# (void) send(receiver, message)

Sends message to an actor

Examples:

How to send a string message 

myActor.send('otherActor', 'myMessage')

How to send an object message 

myActor.send('otherActor', {hello:'hello',
                               obj:{
                                 count:1
                               }
                             })

Parameters:

  • receiver ( String ) — the receiver id
  • message ( Object ) — the content of the message (it could be any js object)

# (void) attachRoute(routePattern)

attach a named route as a function. So if you use this.attachRoute('someRoute'), you can use this.someRoute(someMessage) which is equivalent to this.send('someRoute',someMessage)

Examples:

How to apply a filter transformation 

myActor.attachRoute('someRoute');
myActor.attachRoute(['someRoute1','someRoute2']);
myActor.attachRoute(/some/g);

Parameters:

  • routePattern ( Object ) — it can be a string, an array of routes to be attached or a regular expression of routes
